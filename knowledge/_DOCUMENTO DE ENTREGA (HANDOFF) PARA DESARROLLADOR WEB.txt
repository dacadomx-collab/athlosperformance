1. RESUMEN EJECUTIVO PARA EL EQUIPO DE DESARROLLO
Visión Estratégica: Athlos Performance BCS no es un gimnasio convencional; se posiciona como el primer "Laboratorio de Ciencias del Deporte y Movimiento Humano" en La Paz. El objetivo del sitio es proyectar autoridad médica, rigor empírico y transparencia de datos, alejándose de los clichés del fisicoconstructivismo (prohibido usar imágenes de gimnasio tradicionales). Metodología Científica: La operación se basa en la "Fricción Cero", integrando el Athlos Cognitive Engine v1.0 (un asistente virtual IA 24/7) que automatiza la captación y el filtrado de perfiles. La intervención física se diseña exclusivamente a partir del análisis de datos (antropometría, somatocarta, Senior Fitness Test) obtenidos en las Fichas de Evaluación.
Directrices de UI/UX (Sistema de Diseño Base)
* Tipografías: Montserrat o Sora para encabezados (innovación y precisión); Inter para cuerpos de texto (legibilidad premium).
* Paleta de Colores:

   * Elementos Secundarios/Subtítulos: Azul Acero (#3F6E8A).
   * Botones (CTA): Turquesa Tecnológico (#00B8C9).
   * Fondos de Bloques de Texto: Gris Claro (#F7FAFC).
* Restricción Técnica: Diseño 100% Mobile-First. Prohibido usar anchos fijos en contenedores.

<Section 4: El Motor del Diagnóstico (Uso de Fichas de Evaluación)>
* UI Notes: Esta sección explica cómo se personaliza el servicio. Dividir visualmente la pantalla en dos columnas: Izquierda (Atletas), Derecha (Adultos Mayores).
[Copy Definitivo] Título (H2): El Punto de Partida: Tu Ficha de Evaluación Clínica Cuerpo de texto: Nuestro rigor empírico significa que ninguna decisión de carga física se toma por intuición. Utilizamos protocolos internacionales estandarizados para mapear tu perfil antes de tu primer entrenamiento:
[Columna 1: Atletas y Rendimiento (Menores de 65)] Analizamos tu estructura desde el interior. Calculamos tu densidad corporal, porcentaje de grasa exacta (Fórmula de Siri), masa ósea (Fórmula de Rocha) y te ubicamos en una Somatocarta matemática. Adicionalmente, evaluamos tus patrones de movimiento (como la compensación de rodillas en sentadilla profunda) para asegurar un entrenamiento libre de fricción articular.
[Columna 2: Longevidad Activa (Mayores de 65)] Evaluamos los biomarcadores que garantizan tu independencia. Aplicamos el Senior Fitness Test, midiendo fuerza del tren inferior (Chair Stand), agilidad, equilibrio dinámico y riesgo de caída mediante la prueba Timed Up & Go (TUG) con carga cognitiva. Un segundo de mejora en estas pruebas transforma tu seguridad y calidad de vida.

⚙️ Notas para el Desarrollador (Integración Backend):
1. Formularios/CTAs: Todos los botones primarios deben lanzar el widget/formulario conectado a la base de datos athlos_engine_db vía la API Webhook en api/webhook_mensajeria.php.
2. Captación de Datos (Onboarding): El NLP interceptará los datos solicitando 3 variables iniciales: Nombre Completo, Teléfono normalizado (Lada 612) y Objetivo de Salud, antes de ejecutar la deduplicación y crear el registro del lead en el CRM.
3. Seguridad (Consent Gate): Es obligatorio implementar un Consent Gate de privacidad antes de procesar los datos clínicos en la base de datos para cumplir con la legalidad de datos sensibles (Mandamiento REGLA-01).
